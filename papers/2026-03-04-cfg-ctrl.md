# CFG-Ctrl: Control-Based Classifier-Free Diffusion Guidance

## 基本情報
- **arXiv**: https://arxiv.org/abs/2603.03281
- **著者**: Hanyang Wang, Yiyang Liu, Jiawei Chi, Fangfu Liu, Ran Xue, Yueqi Duan
- **採択**: CVPR 2026
- **Project**: https://hanyang-21.github.io/CFG-Ctrl

## 概要
Classifier-Free Guidance (CFG)を制御理論の観点から再解釈する統一フレームワーク。従来CFGは線形制御に依存し、大きなguidance scaleで不安定性、オーバーシュート、セマンティック忠実度の低下を引き起こす。非線形フィードバック制御（Sliding Mode Control）を導入。

## 主要貢献
1. **CFG as control**: CFGを1次連続時間生成フローへの制御として再解釈
2. **SMC-CFG (Sliding Mode Control CFG)**: 非線形フィードバック誘導補正
3. **Exponential sliding mode surface**: セマンティック予測誤差に対するスライディングモード曲面
4. **Lyapunov stability analysis**: 有限時間収束の理論的証明

## 実験結果
- Stable Diffusion 3.5、Flux、Qwen-Imageで検証
- 標準CFGを上回るセマンティックアライメント
- 広範囲のguidance scaleでロバスト性向上
