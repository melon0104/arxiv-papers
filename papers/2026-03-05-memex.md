# Memex(RL): Scaling Long-Horizon LLM Agents via Indexed Experience Memory

**arXiv**: [2603.04257](https://arxiv.org/abs/2603.04257)

## 著者
Zhenting Wang, Huancheng Chen, Jiayun Wang, Wei Wei

## 一言まとめ
長期間タスク向けのインデックス付きメモリ機構。要約のみの手法より大幅に低損失なメモリを実現し、MemexRLで書き込み・読み取り動作を最適化。

## 概要
LLMエージェントは有限コンテキストウィンドウにより長期間タスクでボトルネック。既存の切り捨て・要約手法は過去の証拠を圧縮・破棄するため損失が大きい。

### 主要な技術貢献
1. **Indexed Experience Memory (Memex)**: コンパクトな作業コンテキスト（簡潔な要約+安定インデックス）を維持
2. **External experience database**: 完全忠実度のインタラクションを外部保存
3. **MemexRL**: コンテキスト予算下でのインデックスメモリ使用に特化した報酬整形

### 理論・実験結果
- 境界付きデリファレンスで意思決定品質を保持（理論保証）
- 履歴増加時も実効コンテキスト内計算を有界に維持
- 長期間タスクでタスク成功率向上、作業コンテキスト大幅削減
