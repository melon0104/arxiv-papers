# SORT: A Systematically Optimized Ranking Transformer for Industrial-scale Recommenders

**arXiv**: [2603.03988](https://arxiv.org/abs/2603.03988)

## 著者
Chunqi Wang, Bingchao Wu, Taotian Pang, Jiahao Wang, Jie Yang, Jia Liu, Hao Zhang, Hai Zhu, Lei Shen, Shizhun Wang, Bing Wang, Xiaoyi Zeng

## 一言まとめ
産業規模のランキングモデルにTransformerを適用し、大規模Eコマースで**GMV +5.47%、注文数 +6.35%**を達成。レイテンシ44%削減、スループット121%向上。

## 概要
TransformerはLLMで顕著な成功を収めているが、高いfeature sparsityと低いlabel densityの課題から、産業規模のランキングモデルへの適用は未成熟だった。本論文では以下の最適化を提案：

### 主要な技術貢献
1. **Request-centric sample organization**: リクエスト単位でサンプルを組織化
2. **Local attention + Query pruning**: 計算効率の大幅改善
3. **Generative pre-training**: ラベル不足問題への対処
4. **Tokenization/MHA/FFN最適化**: 学習安定化とモデル容量拡大

### 実験結果
- **オンラインA/Bテスト（大規模Eコマース）**:
  - 注文数: +6.35%
  - 購入者数: +5.97%
  - GMV: +5.47%
  - レイテンシ: -44.67%
  - スループット: +121.33%
- Model FLOPs Utilization (MFU): 22%達成
