# QAQ: Bidirectional Semantic Coherence for Selecting High-Quality Synthetic Code Instructions

## Authors
Jiayin Lei, Ming Ma, Yunxi Duan, Chenxi Li, Tianming Yang

## Abstract
Synthetic data has become essential for training code generation models, yet it introduces significant noise and hallucinations that are difficult to detect with current metrics.

Existing data selection methods like **Instruction-Following Difficulty (IFD)** typically assess how hard a model generates an answer given a query (A|Q). However, this metric is ambiguous on noisy synthetic data, where low probability can distinguish between intrinsic task complexity and model-generated hallucinations.

## The QAQ Framework
We propose **QAQ**, a novel data selection framework that evaluates data quality from the **reverse direction**:

- Traditional: P(Answer | Question)
- QAQ: Also considers P(Question | Answer) - bidirectional semantic coherence

### Key Insight
High-quality instruction-answer pairs should exhibit strong bidirectional coherence:
- Good answers should make the questions recoverable
- Hallucinated or noisy answers break this symmetry

## Results
QAQ provides a more robust signal for filtering synthetic code data, leading to:
- Reduced noise in training data
- Better generalization in code generation models
- More efficient use of synthetic data budgets
