# Simulation Distillation: Pretraining World Models in Simulation for Rapid Real-World Adaptation

## 基本情報

| 項目 | 内容 |
|------|------|
| arXiv | [2603.15759](https://arxiv.org/abs/2603.15759) |
| プロジェクト | https://sim-dist.github.io/ |
| 検証タスク | 精密マニピュレーション、四足歩行ロコモーション |

## 概要

シミュレーションから潜在ワールドモデルに構造的事前知識を蒸留し、オンラインプランニングと教師ありダイナミクス微調整で**高速な実世界適応**を実現するフレームワーク。

## 問題設定

- Sim-to-real転移はロボティクスの中心的課題
- シミュレーションと実世界のダイナミクスミスマッチで失敗
- 既存のファインチューニング手法の課題：
  - 探索の困難さ
  - 長期的クレジット割り当て
  - **低データ環境**での適応

## 提案手法: SimDist

### シミュレーションからの蒸留
- シミュレータの構造的事前知識を**潜在ワールドモデル**に蒸留
- 報酬モデルと価値モデルをシミュレーションから直接転送

### 実世界適応
- **オンラインプランニング** + **教師ありダイナミクス微調整**
- 生の知覚から密なプランニング信号を提供
- 価値学習不要

### 核心の洞察
> 実世界適応を**短期間のシステム同定**に還元

- 長期的クレジット割り当てを回避
- 高速かつ安定した改善

## 主要な結果

| 指標 | 結果 |
|------|------|
| データ効率 | 従来手法を**大幅上回る** |
| 安定性 | 高い |
| 最終性能 | SOTA |

## 技術的貢献

1. Sim-to-realを教師あり学習問題に還元
2. ワールドモデルへの構造蒸留
3. 長期的クレジット割り当て問題の回避

## 意義

Sim-to-real問題への新しいアプローチ。価値学習なしで高速適応を実現し、実用的なロボット学習に貢献。
