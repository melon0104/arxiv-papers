# RSIR: Recursive Self-Improving Recommendation Framework

**arXiv ID:** 2602.15659  
**Published:** 2026-02-17  
**Categories:** cs.IR  
**Code:** https://anonymous.4open.science/r/RSIR-7C5B

## Authors
Luankang Zhang, Hao Wang, Zhongzhou Liu, Mingjia Yin, Yonghao Huang, Jiaqi Li, Wei Guo, Yong Liu, Huifeng Guo, Defu Lian, Enhong Chen

## Summary
推薦システムにおける高品質な学習データの不足問題に取り組む新しいフレームワーク。RSIRは外部データやTeacherモデルなしに、モデル自身が性能を向上させる自己改善パラダイムを提案。

## Key Contributions
- **自己改善ループ**: 現在のモデルがユーザーインタラクションシーケンスを生成し、Fidelity-based品質管理で選別
- **暗黙的正則化**: データ駆動型の最適化ランドスケープの平滑化効果を理論的に証明
- **累積的改善**: 複数のベンチマークとアーキテクチャで一貫した性能向上を確認
- 弱いモデルでも強いモデルの学習カリキュラムを生成可能

## Technical Approach
1. 現行モデルがユーザーの近似的選好多様体と一致するインタラクションシーケンスを生成
2. Fidelityベースのフィルタリングで質の低いデータを除去
3. 強化されたデータセットで後継モデルを学習

## Results
- 複数ベンチマークで累積的・一貫した性能向上
- 小規模モデルでも効果を発揮
- データスパース性問題を解決するモデル非依存なアプローチ

## Significance
推薦システムにおける学習データ不足という根本的なボトルネックに対する新しいソリューション。自己改善という概念は他のML分野にも応用可能。
