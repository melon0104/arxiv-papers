# Recursive Concept Evolution (RCE) for Compositional Reasoning in LLMs

**arXiv ID:** 2602.15725  
**Published:** 2026-02-17  
**Categories:** cs.AI, cs.CL, cs.LG

## Authors
Sarim Chaudhry

## Summary
LLMの構成的推論能力を向上させる新しいフレームワーク。推論中にモデルの内部表現幾何学を動的に変更し、新しい抽象概念を構築可能に。

## Key Contributions
- **問題の特定**: トークンレベル探索の拡張（CoT, self-consistency, RL）では潜在表現空間が固定され、必要な抽象が存在しない場合に性能崩壊
- **RCE**: 推論中に内部表現幾何学を変更可能にするフレームワーク
- **動的概念部分空間**: 表現の不十分さを検出した際にlow-rank概念部分空間を動的生成
- **MDL基準**: Minimum Description Lengthによる部分空間の選択とマージ

## Technical Approach
1. 表現の不十分さを検出
2. low-rank概念部分空間を動的に生成
3. MDL基準で部分空間を選択
4. シナジー効果がある場合はマージ
5. 制約付き最適化で安定性を保持しながら統合

## Results
Mistral-7Bでの評価：
- ARC-AGI-2: 12-18ポイント向上
- GPQA, BBH: 8-14ポイント向上
- MATH, HLE: 深さ誘発エラーの一貫した削減

## Significance
構成的推論における根本的な限界に取り組む野心的な研究。既存抽象の再結合ではなく、新しい抽象の構築を可能にする点が革新的。
