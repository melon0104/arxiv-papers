# GLM-5: from Vibe Coding to Agentic Engineering

**arXiv ID:** 2602.15763  
**Published:** 2026-02-17  
**Categories:** cs.LG, cs.CL  
**Institution:** Tsinghua University (Zhipu AI)  
**Code:** https://github.com/zai-org/GLM-5

## Authors
GLM-5 Team (Aohan Zeng, Xin Lv, Zhengxiao Du, Jie Tang et al.)

## Summary
Vibe CodingからAgentic Engineeringへのパラダイムシフトを実現する次世代基盤モデル。コーディング、推論、エージェント能力（ARC）を大幅に強化。

## Key Contributions
- **DSA（Differential State Aggregation）**: 学習・推論コストを削減しながら長コンテキスト忠実度を維持
- **非同期強化学習インフラ**: 生成と学習を分離して事後学習効率を大幅向上
- **非同期エージェントRLアルゴリズム**: 複雑な長期的インタラクションからの効果的な学習
- 実世界コーディングタスクで前例のない性能

## Technical Approach
1. Agentic, Reasoning, Coding (ARC) 能力の統合
2. DSAによる効率的なアテンション機構
3. 非同期RLによるエージェント能力の強化
4. End-to-endソフトウェアエンジニアリング対応

## Results
- 主要オープンベンチマークでSOTA達成
- 実世界コーディングタスクで既存ベースラインを凌駕
- エンドツーエンドのソフトウェアエンジニアリングチャレンジに対応

## Significance
清華大学からの重要なLLMリリース。エージェント能力とコーディング性能の両立を実現し、AIアシスタントの実用性を大きく前進させる。
