# Dex4D: Task-Agnostic Point Track Policy for Sim-to-Real Dexterous Manipulation

**arXiv ID:** 2602.15828  
**Published:** 2026-02-17  
**Categories:** cs.RO, cs.CV, cs.LG  
**Institution:** CMU (Katerina Fragkiadaki, Shubham Tulsiani)  
**Project:** https://dex4d.github.io/

## Authors
Yuxuan Kuang, Sungjae Park, **Katerina Fragkiadaki**, **Shubham Tulsiani**

## Summary
タスク非依存の3Dポイントトラック条件付きポリシーでSim-to-Real巧緻操作を実現。シミュレーションで学習し、実世界タスクにゼロショット転移。

## Key Contributions
- **Anypose-to-Anyposeポリシー**: 任意の物体を任意の姿勢に操作可能なドメイン非依存ポリシー
- **タスク非依存学習**: 数千の物体・多様な姿勢構成でシミュレーション学習
- **ゼロショット転移**: ファインチューニングなしで実世界タスクに展開
- **オンラインポイントトラッキング**: 閉ループ知覚・制御のためのリアルタイムトラッキング

## Technical Approach
1. シミュレーションで広範なロボット-物体インタラクションをカバーする学習
2. テスト時に生成動画から抽出した物体中心ポイントトラックでポリシーをプロンプト
3. 実行中はオンラインポイントトラッキングで閉ループ制御
4. 構成可能なスキルでテスト時に多様なタスクに対応

## Results
シミュレーションと実ロボットでの広範な実験：
- 多様な巧緻操作タスクでゼロショット展開に成功
- 既存ベースラインに対して一貫した改善
- 新規物体、シーンレイアウト、背景、軌道への強い汎化
- ロバスト性とスケーラビリティを実証

## Significance
CMU（Fragkiadaki, Tulsiani）からの重要なロボティクス研究。タスク固有の報酬設計やテレオペレーションデータ収集という大きな障壁を克服し、巧緻操作のスケーラブルな学習を可能に。
