# Persona2Web: Benchmarking Personalized Web Agents for Contextual Reasoning with User History

- **arXiv ID**: 2602.17003
- **カテゴリ**: cs.CL, cs.AI
- **投稿日**: 2026-02-19
- **リンク**: https://arxiv.org/abs/2602.17003
- **コード**: GitHub公開

## 概要

大規模言語モデルはWebエージェントを進化させてきたが、現在のエージェントにはパーソナライゼーション能力が欠けている。ユーザーは意図の詳細をすべて指定することは稀であり、実用的なWebエージェントはユーザーの好みやコンテキストを推論して曖昧なクエリを解釈できる必要がある。

Persona2Webは、実際のオープンWebでパーソナライズされたWebエージェントを評価する初のベンチマーク。「clarify-to-personalize」原則に基づき、明示的な指示ではなくユーザー履歴から曖昧さを解決することを要求する。(1) 長期間にわたって暗黙的に好みを示すユーザー履歴、(2) 暗黙的なユーザー好みの推論を要求する曖昧なクエリ、(3) パーソナライゼーションの詳細評価を可能にする推論認識型評価フレームワークで構成。
