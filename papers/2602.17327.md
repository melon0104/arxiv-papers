# WebFAQ 2.0: A Multilingual QA Dataset with Mined Hard Negatives for Dense Retrieval

- **arXiv ID**: 2602.17327
- **カテゴリ**: cs.IR, cs.AI, cs.CL
- **投稿日**: 2026-02-19
- **リンク**: https://arxiv.org/abs/2602.17327
- **コード**: GitHub, HuggingFace公開

## 概要

WebFAQ 2.0は、108言語にわたる1億9800万のFAQベースの自然な質問-回答ペアを含む新バージョンのデータセット。前バージョンと比較して、多言語カバレッジを大幅に拡張し、バイリンガルアラインメントQAペアを1430万以上に増加させ、最大のFAQベースリソースとなった。

新しいデータ収集戦略により、Webコンテンツを直接クロール・抽出し、ページタイトルと説明を通じてより豊富なコンテキストを持つ多様で多言語なデータセットを実現。また、20言語125万クエリのハードネガティブデータセットも公開し、Contrastive LearningとKnowledge Distillationの両方の学習戦略を可能にする。

## 注目ポイント

- **大規模多言語リソース**: 108言語、1.98億QAペア
- **ハードネガティブ公開**: 密検索器の学習に直接利用可能
- **継続的更新**: Open Web Indexを通じて定期的にFAQが追加
- **オープンソース**: GitHub, HuggingFace公開
