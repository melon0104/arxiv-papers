# Modeling Distinct Human Interaction in Web Agents

**arXiv**: [2602.17588](https://arxiv.org/abs/2602.17588)

## 概要

自律型Webエージェントにおける人間介入のモデル化。いつ・なぜユーザーが介入するかを原理的に理解し、協調的タスク実行を支援。

## 主要な貢献

1. **CowCorpus**: 400件の実ユーザーWeb操作軌跡、4,200+の人間・エージェント交互アクション
2. **4つの介入パターン**の特定:
   - Hands-off supervision（放任監督）
   - Hands-on oversight（積極監視）
   - Collaborative task-solving（協調解決）
   - Full user takeover（完全引き継ぎ）
3. **介入予測モデル**: ユーザースタイルに基づく介入タイミング予測

## 実験結果

- 介入予測精度: ベースLMより**61.4-63.4%改善**
- ユーザー評価: エージェント有用性**26.5%向上**
- 追加レイテンシ: 30ms未満/インタラクション

## 意義

エージェントの「自律しすぎ」「確認しすぎ」問題を解決。人間-AI協調の新しい設計指針を提供。
