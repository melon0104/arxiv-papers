# M-Attack-V2: Pushing the Frontier of Black-Box LVLM Attacks via Fine-Grained Detail Targeting

- **arXiv ID**: 2602.17645
- **カテゴリ**: cs.LG, cs.AI, cs.CL, cs.CV
- **投稿日**: 2026-02-19
- **リンク**: https://arxiv.org/abs/2602.17645
- **コード**: https://github.com/vila-lab/M-Attack-V2

## 概要

大規模ビジョン言語モデル（LVLM）へのブラックボックス敵対的攻撃は、勾配の欠如と複雑なマルチモーダル境界のため困難。先行手法M-Attackのローカルクロップレベルマッチングは高分散でほぼ直交する勾配を引き起こし、一貫したローカルアラインメントに違反し最適化を不安定にする問題があった。

本研究では、ローカルマッチングをソース変換とターゲットセマンティクスに対する非対称期待値として再定式化。Multi-Crop Alignment (MCA)、Auxiliary Target Alignment (ATA)、Patch Momentumなどのモジュールを組み合わせ、M-Attack-V2を構築。Claude-4.0で成功率を8%から30%に、Gemini-2.5-Proで83%から97%に、GPT-5で98%から100%に向上させ、先行ブラックボックスLVLM攻撃を上回る。

## 注目ポイント

- **フロンティアモデルへの攻撃**: GPT-5、Claude-4.0、Gemini-2.5-Proをターゲット
- **大幅な成功率向上**: 特にClaude-4.0で8%→30%
- **モジュラー設計**: M-Attackへの段階的拡張
- **コード公開**: GitHub
