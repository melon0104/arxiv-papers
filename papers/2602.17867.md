# ADAPT: Hybrid Prompt Optimization for LLM Feature Visualization

## 基本情報
- **arXiv**: [2602.17867](https://arxiv.org/abs/2602.17867)
- **カテゴリ**: cs.LG, cs.CL
- **投稿日**: 2026-02-19

## 概要
LLMの活性化空間で学習された方向がエンコードする特徴を理解するため、ターゲット方向を最大活性化する入力を同定する特徴可視化手法「ADAPT」を提案。

## 主要な貢献
1. **ハイブリッド手法**: ビームサーチ初期化と適応的勾配誘導突然変異を組み合わせ
2. **局所最適対策**: テキストの離散性による局所最適に陥りやすい問題に対処
3. **厳密な評価指標**: データセット活性化統計に基づくメトリクスを提案

## 技術的詳細
- Gemma 2 2BのSparse Autoencoder潜在表現を評価
- 層とlatentタイプ全体で従来手法を一貫して上回る
- 特徴可視化がLLMで実現可能であることを確立

## 意義
- CNNで成功した特徴可視化をLLMに拡張
- ドメイン固有の設計前提の重要性を示す
- Mechanistic Interpretabilityへの新アプローチ

## 関連キーワード
Feature Visualization, Sparse Autoencoder, LLM Interpretability, Prompt Optimization
