# NovaPlan: Zero-Shot Long-Horizon Manipulation via Closed-Loop Video Language Planning

## 概要

**閉ループVLM・ビデオプランニング**と幾何学的に接地されたロボット実行を統合する階層的フレームワーク。ゼロショット長期視野操作を実現。

## 主な貢献

- **NovaPlan**: 高レベル意味推論と低レベル物理的相互作用を統合
- 事前デモンストレーションや訓練なしで複雑組立タスクを実行
- **CMU Kris Kitani**、**Brown George Konidaris**（著名ロボティクス研究者）

## 技術詳細

### 階層的設計

#### 高レベル: VLMプランナー
- タスクをサブゴールに分解
- ロボット実行を閉ループで監視
- 単一ステップ失敗からの自律的再プランニング

#### 低レベル: ロボットアクション計算
1. 生成動画からタスク関連オブジェクトキーポイントを抽出
2. 人間の手姿勢を運動学的事前分布として活用
3. **切り替えメカニズム**: より良い参照を選択
4. 重度オクルージョンや深度不正確さでも安定実行

### 実験結果
- 3つの長期視野タスクで有効性実証
- Functional Manipulation Benchmark (FMB)で評価
- **事前デモなしで**複雑組立・器用なエラー回復
