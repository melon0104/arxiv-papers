# MoBiQuant: Mixture-of-Bits Quantization

## 基本情報
- **arXiv**: [2602.20191](https://arxiv.org/abs/2602.20191)
- **カテゴリ**: cs.LG, cs.AI, cs.CL
- **著者**: Duke University（Yiran Chen）、Panasonic R&D他

## 概要
クラウド・エッジデバイスでの変動する実行時複雑さに対応する弾性LLMデプロイメントを実現。トークン感度に基づき重み精度を調整するMixture-of-Bits量子化フレームワークを提案。

## 主要な貢献
- 精度依存のアウトライヤー移行現象によるトークンレベル感度の変動を特定
- **Many-in-one recursive residual quantization**: 高精度重みを反復的に再構築
- **Token-aware router**: 残差ビットスライス数を動的選択
- トークンアウトライヤー分布への汎化を改善しつつスムーズな精度切り替えを実現

## 技術的詳細
- Recursive residual quantization: 低精度から高精度への反復再構築
- Token sensitivity: 精度依存のアウトライヤー移行に起因
- Dynamic bit-width selection: 推論時のリソースに応じた適応

## 結果
LLaMA3-8BでビットSpecific calibrated PTQと同等性能を、反復キャリブレーションなしで達成
