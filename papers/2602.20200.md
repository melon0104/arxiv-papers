# OptimusVLA: Dual-Memory VLA Framework

## 基本情報
- **arXiv**: [2602.20200](https://arxiv.org/abs/2602.20200)
- **カテゴリ**: cs.RO, cs.AI, cs.CV

## 概要
階層的Vision-Language-Action（VLA）モデルの行動生成プロセスのボトルネック（低推論効率、低ロバスト性）を解決。Global Prior MemoryとLocal Consistency Memoryによるデュアルメモリフレームワークを提案。

## 主要な貢献
- **Global Prior Memory (GPM)**: セマンティック類似軌道からタスクレベル事前分布を取得し、生成パスを短縮
- **Local Consistency Memory (LCM)**: 実行済みアクションシーケンスを動的にモデル化し、タスク進捗推論と時間的一貫性制約を注入
- ガウシアンノイズを置換することでNFE（関数評価回数）を削減
- 軌道の時間的一貫性と滑らかさを強制

## 結果
- **LIBERO**: 平均成功率98.6%
- **CALVIN**: π_0比13.5%改善
- **RoboTwin 2.0 Hard**: 平均成功率38%
- **実ロボット評価**: Generalization/Long-horizonでπ_0比42.9%/52.4%改善、2.9倍高速化

## 注目ポイント
- 3ベンチマーク+実ロボットでの包括的評価
- 推論速度と性能の両立
- VLAの実用化に向けた重要な進展
