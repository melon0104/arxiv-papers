# IPOW: Interpretable Open-World Object Detection

## 基本情報
- **arXiv**: [2602.20616](https://arxiv.org/abs/2602.20616)
- **カテゴリ**: cs.CV, cs.LG

## 概要
オープンワールド物体検出（OWOD）は、既知カテゴリの増分検出と未知オブジェクトの信頼性高い識別を要求。本研究では、OWODフレームワーク全体を解釈可能にする「IPOW」を提案。

## 主要な貢献
- **Concept Decomposition Model (CDM)**: Faster R-CNNの結合RoI特徴を識別的・共有・背景概念に明示的に分解
- **識別的概念**: 既知カテゴリ間距離を最大化する最も識別的な特徴を特定
- **共有・背景概念**: 強い汎化能力により未知カテゴリ検出に転移
- **Concept-Guided Rectification (CGR)**: 既知-未知混同を解決

## 技術的詳細
- Discriminative concepts: 既知クラス間の識別
- Shared concepts: ドメイン間で共有される特徴
- Background concepts: 背景領域の特徴
- 未知物体が既知クラスの識別空間に入る問題を特定・解決

## 結果
未知リコールを大幅改善しつつ混同を軽減、既知・未知予測の概念レベル解釈可能性を提供

## 注目ポイント
- OWODの解釈可能性という新しい観点
- 既知-未知混同の原因究明と解決
- 実世界応用への重要な一歩
