# SibylSense: Adaptive Rubric Learning

## 基本情報
- **arXiv**: [2602.20751](https://arxiv.org/abs/2602.20751)
- **カテゴリ**: cs.CL, cs.AI, cs.LG
- **著者**: Microsoft Research（Emre Kıcıman, Ranveer Chandra他）、UCLA

## 概要
オープンエンド生成のための整合的でロバストな報酬設計の課題に取り組む。推論時学習アプローチ「SibylSense」を提案し、検証済みルーブリック項目のチューナブルメモリバンクを通じてルーブリック生成器を適応。

## 主要な貢献
- Memory tuning: 検証器ベースの項目報酬（参照-候補回答の識別ギャップ）でメモリ更新
- Rubric-adversarial policy update: ルーブリック満足候補回答を生成し、識別ギャップを縮小
- Retrospective reflection: 新しい品質次元を継続的にキャプチャ
- 2つのオープンエンドタスクで、静的・非適応ベースラインを上回る識別的ルーブリック生成とRL性能向上を達成

## 技術的詳細
- Frozen rubric generator + tunable memory bank
- Verifier-based item rewards: reference-candidate discriminative gaps
- Memory-policy alternating optimization
