# LoDADA: Localized Dynamics-Aware Domain Adaptation

## 基本情報
- **arXiv**: [2602.21072](https://arxiv.org/abs/2602.21072)
- **カテゴリ**: cs.LG, cs.AI, cs.RO

## 概要
Off-dynamics offline RLにおいて、限られたターゲットデータと豊富なソースデータ（異なる遷移ダイナミクス）からポリシーを学習。局所的なダイナミクスミスマッチを活用した新手法を提案。

## 主要な貢献
- **LoDADA（Localized Dynamics-Aware Domain Adaptation）**: 局所的ダイナミクスミスマッチを活用してソースデータを効果的に再利用
- ソース・ターゲットデータセットの遷移をクラスタリングし、ドメイン識別によりクラスタレベルのダイナミクス不一致を推定
- 小不一致クラスタのソース遷移は保持、大不一致クラスタは除外
- 細粒度でスケーラブルなデータ選択戦略

## 技術的詳細
- Cluster-level dynamics discrepancy estimation via domain discrimination
- Fine-grained data selection: サンプル単位ではなくクラスタ単位で効率化
- グローバル仮定の回避と計算コスト削減の両立

## 結果
多様なグローバル・ローカルダイナミクスシフト環境で、最先端off-dynamics offline RL手法を一貫して上回る
