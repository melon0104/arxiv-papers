# Field-Theoretic Memory for AI Agents

## 基本情報
- **arXiv ID**: 2602.21220
- **タイトル**: Field-Theoretic Memory for AI Agents: Continuous Dynamics for Context Preservation
- **カテゴリ**: cs.CL, cs.AI, cs.LG
- **コード**: https://github.com/rotalabs/rotalabs-fieldmem

## 概要
AIエージェント向けメモリシステムを提案。従来のデータベース方式ではなく、偏微分方程式で支配される連続場として情報を扱う。古典場理論からの発想で、メモリはセマンティック空間を拡散し、重要度に基づいて熱力学的に減衰し、マルチエージェントシナリオでは場結合を通じて相互作用。

## 評価ベンチマーク
1. **LoCoMo (ACL 2024)**: 300ターン会話、35セッション
2. **LongMemEval (ICLR 2025)**: 500+ターンのマルチセッション推論

## 実験結果（LongMemEval）
- マルチセッション推論: **+116% F1** (p<0.01, d=3.06)
- 時間的推論: **+43.8%** (p<0.001, d=9.21)
- 知識更新の検索リコール: **+27.8%** (p<0.001, d=5.00)
- マルチエージェント実験: 場結合による**99.8%超の集合知**
