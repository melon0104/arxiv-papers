# Latent Context Compilation

## 基本情報
- **arXiv ID**: 2602.21221
- **タイトル**: Latent Context Compilation: Distilling Long Context into Compact Portable Memory
- **カテゴリ**: cs.LG, cs.AI, cs.CL

## 概要
効率的な長文脈LLMデプロイは、償却圧縮（OOD汎化に弱い）とTest-Time Training（合成データコストと重み変更が必要）のジレンマに直面。本研究はLatent Context Compilation（潜在文脈コンパイル）を提案し、文脈処理を適応からコンパイルへ根本的に転換。

## 主な貢献
1. **使い捨てLoRAコンパイラ**: 長文脈をコンパクトなバッファトークンに蒸留
2. **ステートレス・ポータブルメモリ**: frozen base modelとプラグアンドプレイ互換
3. **自己整合最適化**: 合成QAペア不要の戦略

## 技術的アプローチ
- 文脈再構成タスクを文脈非依存ランダムクエリで正則化
- 圧縮トークンをモデルの既存指示追従多様体内に配置

## 実験結果（Llama-3.1-8B）
- **16倍圧縮率**でも細粒度詳細と推論能力を維持
- 従来手法が失敗する場面でも性能維持
- メモリ密度をモデルパラメータから分離
