# Make Every Draft Count: Hidden State based Speculative Decoding

## 基本情報
- **arXiv ID**: 2602.21224
- **タイトル**: Make Every Draft Count: Hidden State based Speculative Decoding
- **カテゴリ**: cs.CL, cs.AI, cs.DC, cs.LG

## 概要
Speculative decodingはLLM推論高速化の重要な技術だが、ドラフトトークンの多くが検証に失敗して破棄され、計算の無駄が発生。本研究は破棄されたドラフトを再利用可能なトークンに変換するシステムを提案。

## 主な貢献
1. **Hidden state自己回帰予測**: トークン情報の統合を隠れ状態生成後に延期
2. **効率的トークン情報注入**: 高品質ドラフトトークンツリー構築
3. **検証失敗からのトークン再サンプリング**: 隠れ状態の再利用を可能に

## 技術的アプローチ
- Hidden stateレベルで自己回帰予測を実行
- トークンベースドラフターより豊富なセマンティクスを保持
- ハードウェア利用率を最大化するオーバーヘッド排除設計

## 実験結果
- 標準speculative decodingと比較して**最大3.3倍の高速化**
- 複数のベースラインに対する包括的な評価
