# NEO: A Unified Language Model for Large Scale Search, Recommendation, and Reasoning

- **arXiv ID**: 2603.17533
- **Date**: 2026-03-18
- **Category**: cs.IR
- **URL**: https://arxiv.org/abs/2603.17533

## Authors
Marco De Nadai, Edoardo D'Amico, Max Lefarov, et al. (Spotify Research)

## Abstract
LLMs are increasingly applied to recommendation, retrieval, and reasoning, yet deploying a single end-to-end model that can jointly support these behaviors over large, heterogeneous catalogs remains challenging. Such systems must generate unambiguous references to real items, handle multiple entity types, and operate under strict latency and reliability constraints requirements that are difficult to satisfy with text-only generation. While tool-augmented recommender systems address parts of this problem, they introduce orchestration complexity and limit end-to-end optimization.

We introduce NEO, a framework that adapts a pre-trained decoder-only LLM into a tool-free, catalog-grounded generator. NEO represents items as SIDs (Semantic IDs) and trains a single model to interleave natural language and typed item identifiers within a shared sequence. Text prompts control the task, target entity type, and output format (IDs, text, or mixed), while constrained decoding guarantees catalog-valid item generation without restricting free-form text.
