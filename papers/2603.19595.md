# All-Mem: Agentic Lifelong Memory via Dynamic Topology Evolution

## 概要
数ヶ月〜数年にわたるライフロングインタラクティブエージェント向けのオンライン/オフラインメモリフレームワーク。要約ベースの圧縮による不可逆情報損失を回避し、トポロジー構造化メモリバンクを維持。

## 主要貢献
1. **明示的・非破壊的統合**: 要約ベース圧縮の情報損失を回避
2. **Bounded Visible Surface**: オンライン操作で粗い検索コストを固定
3. **3オペレーター編集**: LLM診断者がSPLIT/MERGE/UPDATEをゲーティング実行
4. **Typed Links**: 型付きリンクでホップ制限付きバジェット拡張

## 実験結果
LOCOMO・LONGMEMEVALで代表的ベースラインより改善。

## 注目ポイント
🔧 **エージェント長期記憶** - 実用的なAIエージェントのメモリ課題に直接対応
