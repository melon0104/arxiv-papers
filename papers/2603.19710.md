# AIGQ: An End-to-End Hybrid Generative Architecture for E-commerce Query Recommendation

## 概要
TaobaoホームページのHintQ（事前検索クエリ推薦）に本番導入された初のエンドツーエンド生成フレームワーク。従来のID-based手法の浅いセマンティクス、コールドスタート問題、低い偶発性を解決する。

## 主要貢献
1. **Interest-Aware List SFT (IL-SFT)**: セッション認識行動集約と興味ガイド再ランキングでユーザー意図を忠実にモデル化
2. **IL-GRPO**: 個別クエリ関連性と全体リスト特性を同時最適化する双成分報酬メカニズム
3. **ハイブリッドオフライン-オンラインアーキテクチャ**: AIGQ-DirectとAIGQ-Think（推論強化型）の組み合わせ

## 実験結果
Taobao大規模A/Bテストで主要ビジネス指標に大幅な改善を実証。

## 注目ポイント
