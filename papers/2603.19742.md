# Dual Path Attribution: Efficient Attribution for SwiGLU-Transformers through Layer-Wise Target Propagation

## 概要
SwiGLU Transformer向けの忠実で効率的なアトリビューション手法。凍結トランスフォーマー上で1回のforward＋1回のbackwardパスで情報フローを追跡。

## 主要貢献
1. **Dual Path Attribution (DPA)**: 反事実例不要で忠実なアトリビューション
2. **SwiGLU分解・線形化**: 計算構造を異なる経路に分解
3. **O(1)時間複雑性**: モデルコンポーネント数に対して定数時間

## 実験結果
標準解釈性ベンチマークでSoTA忠実性と前例のない効率性を達成。

## 注目ポイント
🔍 **LLM解釈性** - 大規模モデルの内部機構理解に貢献
