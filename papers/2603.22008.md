# On the Challenges and Opportunities of Learned Sparse Retrieval for Code

## 概要
大規模コードベースの検索はLLMベースソフトウェアエンジニアリングシステムの重要コンポーネント。既存手法は主に密ベクトルモデルに依存し、学習型疎検索（LSR）はコード領域でほぼ未開拓。本研究はコードへの疎検索適用の課題を分析。

## 主要課題
1. **Subword Fragmentation**: サブワードトークン化によるコード構造の断片化
2. **Semantic Gap**: 自然言語クエリとコード間の意味的ギャップ
3. **Language Diversity**: プログラミング言語・サブタスクの多様性
4. **Document Length**: コード文書の長さが疎性とレイテンシに悪影響

## 技術的アプローチ
- 疎検索のコード領域への適用可能性を系統的に分析
- 密検索との比較実験
- 疎・密ハイブリッドの可能性を探索

## 注目ポイント
💻 **コード検索** - RAGやコード補完の基盤技術への洞察
⚡ **疎検索** - 効率性と解釈性を両立する検索パラダイム
