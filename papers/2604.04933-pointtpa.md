# PointTPA: Dynamic Network Parameter Adaptation for 3D Scene Understanding

**arXiv ID:** 2604.04933  
**Category:** cs.CV  
**Published:** 2026-04-06

## Authors
Siyuan Liu, Chaoqun Zheng, Xin Zhou

## Abstract
Scene-level point cloud understanding remains challenging due to diverse geometries, imbalanced category distributions, and highly varied spatial layouts. Existing methods improve object-level performance but rely on static network parameters during inference, limiting their adaptability to dynamic scene data.
