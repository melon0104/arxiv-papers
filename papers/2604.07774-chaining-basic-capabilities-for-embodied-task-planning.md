# Chaining Basic Capabilities for Embodied Task Planning

**arXiv ID:** 2604.07774  
**Category:** cs.RO  
**Published:** 2026-04-09

## Authors
Xu, Peiran, Yin, Yufei, Liu, Hao, Zhou, Yu, Wang, Yutong

## Abstract
RoboAgent is a capability-driven planning pipeline that composes basic vision-language capabilities with a scheduler, all implemented in a single VLM. It is trained via behavior cloning, DAgger, and RL and improves embodied planning benchmarks. Code: https://github.com/woyut/RoboAgent_CVPR26.

## Key Contributions
- 複数の基本能力をスケジューラで連鎖させるRoboAgentを提案。
- BC→DAgger→RLの多段訓練で長期計画を強化。
- CVPR 2026、コード公開。
