# Filling the Gaps: Selective Knowledge Augmentation for LLM Recommenders

**arXiv ID:** 2604.07825  
**Category:** cs.IR  
**Published:** 2026-04-09

## Authors
Lee, Jaehyun, Park, Jinhyung, Lee, Haechan, Lee, Jaewoo

## Abstract
We propose KnowSA_CKP, a selective knowledge augmentation method for LLM-based recommenders. It probes internal knowledge to inject external information only when needed, improving accuracy and context efficiency on real-world datasets. Accepted to SIGIR 2026.

## Key Contributions
- LLM内在知識を比較プローブで推定し、必要箇所だけ知識注入。
- 冗長な情報付与を避け、精度とコンテキスト効率を改善。
- SIGIR 2026採択。
