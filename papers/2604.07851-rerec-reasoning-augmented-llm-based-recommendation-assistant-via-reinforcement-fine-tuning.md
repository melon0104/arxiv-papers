# ReRec: Reasoning-Augmented LLM-based Recommendation Assistant via Reinforcement Fine-tuning

**arXiv ID:** 2604.07851  
**Category:** cs.IR  
**Published:** 2026-04-09

## Authors
Huang, Jiani, Wang, Jiacheng, Ma, Zixiang, Zhao, Jian, Liu, Shiqian

## Abstract
ReRec is a reinforcement fine-tuning framework for LLM-based recommendation assistants. It introduces dual-graph reward shaping, reasoning-aware advantage estimation, and an online curriculum scheduler to improve multi-step reasoning while preserving instruction-following. Code: https://github.com/jiani-huang/ReRec.

## Key Contributions
- 推薦タスク向けにRFTを設計し、推論セグメントを明示的に最適化。
- Dual-Graph報酬とカリキュラム学習で推薦精度と推論安定性を両立。
- ACL 2026採択、コード公開。
