# Task-Adaptive Retrieval over Agentic Multi-Modal Web Histories via Learned Graph Memory

**arXiv ID:** 2604.07863  
**Category:** cs.IR  
**Published:** 2026-04-09

## Authors
Forouzandeh, Saman, Li, Jing, Mi, Xinyu, Wang, Jing, Chai, Yekun, Khandelwal, Urvashi, Chen, Wenhu

## Abstract
We propose ACGM, a learned graph-memory retriever that builds task-adaptive relevance graphs over multimodal agent histories. It learns modality-specific decay and sparse connectivity, enabling efficient O(log T) retrieval and improving nDCG/Precision on WebShop, VisualWebArena, and Mind2Web. Code is available at https://github.com/S-Forouzandeh/ACGM-Agentic-Web.

## Key Contributions
- エージェントの長期Web履歴からタスク適応の関連グラフを構築するACGMを提案。
- モダリティ別の時間減衰と疎結合で効率的な履歴検索を実現。
- WebShop/VisualWebArena/Mind2Webで検索精度を大幅改善（コード公開）。
