# Efficient Provably Secure Linguistic Steganography via Range Coding

**arXiv ID:** 2604.08052  
**Category:** cs.CL  
**Published:** 2026-04-09

## Authors
Yan, Ruiyi, Cai, Shu, Liang, Juming, He, Di, Pan, Yali, Wang, Qiang

## Abstract
We propose a provably secure linguistic steganography method using range coding with rotation, achieving near-100% entropy utilization and high embedding speed. Code: http://github.com/ryehr/RRC_steganography. ACL 2026 Main.

## Key Contributions
- Range coding＋回転機構で高効率な言語ステガノグラフィを実現。
- 埋め込み効率ほぼ100%と高スループットを達成。
- ACL 2026 Main、コード公開。
