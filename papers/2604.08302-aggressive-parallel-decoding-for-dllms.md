# Aggressive Parallel Decoding for dLLMs

**arXiv ID:** 2604.08302  
**Category:** cs.LG  
**Published:** 2026-04-09

## Authors
Chen, Zigeng, Dai, Yuan, Yu, Neng, Xiao, Wen

## Abstract
DMax enables aggressive parallel decoding for diffusion language models by progressive self-refinement from mask to token embeddings. On-Policy Uniform Training and Soft Parallel Decoding boost throughput while preserving quality. Code: https://github.com/czg1225/DMax.

## Key Contributions
- マスク→トークン埋め込みへの自己精錬で並列デコード誤差を抑制。
- On-Policy Uniform TrainingとSoft Parallel DecodingでTPSを大幅向上。
- コード公開。
