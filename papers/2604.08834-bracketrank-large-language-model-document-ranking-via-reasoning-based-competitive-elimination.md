# BracketRank: Large Language Model Document Ranking via Reasoning-based Competitive Elimination

- **arXiv ID**: 2604.08834
- **カテゴリ**: cs.IR
- **投稿日**: 2026-04-10
- **URL**: https://arxiv.org/abs/2604.08834

## 著者
Abdelrahman Abdallah, Mohammed Ali, Bhawna Piryani, Adam Jatowt

## 選定理由
✅ **トップ会議採択**: ACL 2026 Main

## 概要
LLM再ランキングの順序感度と文脈制約を緩和するため、推論ベースの競争的淘汰でランキングを構成する。

## 主要な貢献
1. **Reasoning-based competitive elimination**: ペア/小集合で推論し候補を淘汰
2. **順序感度の低減**: 入力順に依存しないランキング生成
3. **長文制約の緩和**: 局所比較でコンテキストを節約

## 実験結果
推論型検索における再ランキング精度の改善を報告。
