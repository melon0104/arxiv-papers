## 概要
部分観測のマルチエージェント協調で、意思決定価値を直接最適化する通信学習を提案。

## 主要な貢献
1. **Value-aware communication**: 決定価値に直結する情報交換
2. **Sequential coordination**: 逐次通信による協調
3. **公開コード**: 再現性確保

## 実験結果
協調タスクで性能向上を報告。
