Hui Liu, Bin Zou, Kecheng Chen, Jie Liu, Wenya Wang, Haoliang Li

## 概要
タスクプロファイルを使ってLLMルーティングを冷スタートから成立させ、コスト/品質の最適化を狙う。

## 主要な貢献
1. **Task-profile guided routing**: タスク特性に基づくモデル選択
2. **Multi-level data synthesis**: 冷スタート用の合成データ生成
3. **コスト-品質最適化**: ルーティング戦略の体系化

## 実験結果
冷スタートでのルーティング性能改善を示す。
