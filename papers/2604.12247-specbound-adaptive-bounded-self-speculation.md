# SpecBound: Adaptive Bounded Self-Speculation with Layer-wise Confidence Calibration

- **arXiv ID**: 2604.12247
- **カテゴリ**: cs.LG
- **投稿日**: 2026-04-14
- **URL**: https://arxiv.org/abs/2604.12247

## 選定理由
✅ **トップ会議採択**: ACL 2026 Findings

## 概要
self-draft型 speculative decoding の弱点である浅層の過信と難トークン混入を、layer-wise calibration と speculation length の適応制御で解決する。

## ポイント
1. **layer-wise temperature annealing** で early-exit の過信を抑制。
2. 難トークンに応じて **speculation length** を制限。
3. base LLM の exact output equivalence を保ちながら最大 2.33x 高速化。
