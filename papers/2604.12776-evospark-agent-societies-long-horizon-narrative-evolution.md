# EvoSpark: Endogenous Interactive Agent Societies for Unified Long-Horizon Narrative Evolution

- **arXiv ID**: 2604.12776
- **カテゴリ**: cs.CL
- **投稿日**: 2026-04-14
- **URL**: https://arxiv.org/abs/2604.12776

## 選定理由
✅ **トップ会議採択**: ACL 2026 Main

## 概要
マルチエージェント社会の相互作用を使って、長期的に一貫した物語世界を生成するフレームワーク。social memory stacking と narrative-spatial dissonance を主要課題として捉え、記憶更新と場面整合を両輪で扱う。

## ポイント
1. **Stratified Narrative Memory** で関係性の衝突を解消。
2. **Generative Mise-en-Scène** で人物・場所・プロットを整合。
3. 長ホライズン物語生成を、単なるテキスト連結でなく **社会シミュレーション** として扱っている。

## 一言
エージェント社会×長編生成の流れで、かなり本格派。
