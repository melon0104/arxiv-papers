# GGD-SLAM: Monocular 3DGS SLAM Powered by Generalizable Motion Model for Dynamic Environments

- **arXiv ID**: 2604.12837
- **カテゴリ**: cs.RO
- **投稿日**: 2026-04-14
- **URL**: https://arxiv.org/abs/2604.12837

## 選定理由
✅ **トップ会議採択**: ICRA 2026

## 概要
静的環境仮定に弱い3DGS系SLAMを、generalizable motion model と動的/静的分離で強化。semantic annotation や depth 入力なしで dynamic scene に対応する。

## ポイント
1. sequential attention を使った **dynamic semantic feature extraction**。
2. occlusion 埋めと distractor-adaptive SSIM loss で静的成分を守る。
3. 実世界の dynamic dataset で camera pose / dense reconstruction を改善。

## 一言
3DGS-SLAMが実環境に寄ってきた感じがする一本。
