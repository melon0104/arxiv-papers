# ROSE: An Intent-Centered Evaluation Metric for NL2SQL

- **arXiv ID**: 2604.12988
- **カテゴリ**: cs.AI
- **投稿日**: 2026-04-14
- **URL**: https://arxiv.org/abs/2604.12988

## 選定理由
✅ **トップ会議採択**: ACL 2026 Main

## 概要
NL2SQL評価で広く使われる Execution Accuracy が、構文差・多義性・誤った正解SQLに弱いことを指摘し、ユーザー意図中心の評価指標 ROSE を提案する。

## ポイント
1. **question intent** を基準に predicted SQL を判定する設計。
2. Prover-Refuter cascade で、ground truth SQL を“証拠”として使うのが巧い。
3. 19手法の再評価から、既存ランキングがかなり怪しいことを示した。

## 一言
NL2SQLの評価をかなり健全化しそう。実運用観点でも重要。
