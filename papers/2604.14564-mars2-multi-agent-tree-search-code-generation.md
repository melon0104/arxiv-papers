# MARS²: Scaling Multi-Agent Tree Search via Reinforcement Learning for Code Generation

**arXiv:** https://arxiv.org/abs/2604.14564
**Published:** 2026-04-17
**Venue:** ACL 2026
**Category:** cs.CL
**Code:** GitHub公開あり

## Authors
Pengfei Li, Shijie Wang, Fangyuan Li, Yikun Fu, et al.

## 概要
強化学習パラダイムはコード生成等の推論集約タスクで強力だが、限定的な軌跡多様性により収穫逓減が生じパフォーマンス上限を制約。探索拡張RLは構造的探索で緩和するが単一エージェントポリシーに制約される。複数の相互作用ポリシーによる多様な探索信号を活用することで、MARS²は複数エージェントのツリー探索をRLでスケールさせる新フレームワークを提案。

## Key Contributions
- **Multi-Agent Tree Search:** 複数エージェントが協調するツリー探索で多様性を大幅向上
- RL最適化との統合により探索と活用を動的バランス
- コード生成での性能上限を大きく引き上げ
- GitHub公開・ACL 2026本会議採択

## Why It Matters
Test-time computeのスケーリングが重要トレンドの中、多エージェント協調ツリー探索は有望な方向性。コード生成に限らず数学推論等への一般化も期待。
