# Federated User Behavior Modeling for Privacy-Preserving LLM Recommendation

**Authors:** Lei Guo, Hongyun Yang, Pengjie Ren, Tong Chen, Hui Liu

**Published:** 2026-04-16

**arXiv:** https://arxiv.org/abs/2604.14833

## 概要

LLMベース推薦システムにおいて、ユーザーデータの少なさ・スパース性が行動モデリングの質を制限している。既存のクロスドメイン推薦（CDR）手法はドメイン間でデータが重複・共有可能であることを前提とするが、プライバシー問題を考慮していない。

本論文は**連合学習（Federated Learning）**を活用し、**プライバシーを保護しつつLLM推薦のためのユーザー行動モデルを構築**するフレームワークを提案する。ドメイン間でのデータ共有なしに、各クライアントのローカルデータのみでLLMの行動把握能力を強化。

## ポイント

- **連合学習 × LLM推薦**：データをローカルに保持したままクロスドメイン知識を活用
- プライバシー保護型クロスドメイン推薦の先駆的アプローチ
- ユーザーデータのスパース性問題を連合型ユーザー行動モデリングで解消
