# GenRec: A Preference-Oriented Generative Framework for Large-Scale Recommendation

**arXiv**: http://arxiv.org/abs/2604.14878v1  
**Category**: cs.IR  
**Published**: 2026-04-16

## 著者
Yanyan Zou, Junbo Qi, Lunsong Huang ほか

## 概要

Generative Retrieval（GR）はNext-Token Predictionで推薦を行う有望パラダイムだが、**産業規模スケールへの適用**には3つの課題があった：

1. **同一リクエストでの出力不一致**（ページネーション機構による）
2. **長い行動列のエンコードコストが高い**（マルチトークンSemantic IDベース）
3. **生成ポリシーと細粒度ユーザー嗜好のズレ**

本論文の **GenRec** はこれら3課題を解決する嗜好指向生成フレームワーク。具体的には：一貫性保証メカニズム、効率的な行動列圧縮、嗜好アライメント損失を統合。

## 主な貢献

1. **産業スケールGRの実用化** - 既存学術手法では不可能だったページネーション問題を解決
2. **効率的エンコーディング** - 長期行動列を計算コストを抑えながら表現
3. **嗜好整合** - 生成分布をユーザーの細粒度選好に近づけるアライメント手法

## なぜ注目？

生成的推薦の産業展開における実践的な問題を正面から扱っており、ECや動画プラットフォームの推薦エンジン開発者には直接参考になる内容。
