# SAGER: Self-Evolving User Policy Skills for Recommendation Agent

**arXiv**: http://arxiv.org/abs/2604.14972v1  
**Category**: cs.IR  
**Published**: 2026-04-16

## 著者
Zhen Tao, Riwei Lai, Chenyun Yu ほか

## 概要

LLMベース推薦エージェントは、ユーザーごとのセマンティックメモリを進化させることでパーソナライズを実現してきたが、**推論ロジック（システムプロンプト）は全ユーザー共通の静的なまま**という根本的な非対称性があった。

本論文では **SAGER**（Self-Evolving User Policy Skills for Recommendation Agent）を提案。推薦が失敗した際、エージェントはユーザー好みのメモリだけでなく、**意思決定ロジック自体を自己修正**する。具体的にはユーザーごとの"Policy Skills"（推論ルール集）を動的に生成・更新し、次回の推薦に活かす。

## 主な貢献

1. **ユーザー固有ポリシースキルの概念化** - 全ユーザー共通の推論ロジックではなく、ユーザーごとに異なる意思決定ルールを動的に管理
2. **自己進化ループ** - 推薦失敗→失敗原因の分析→ポリシースキルの更新→次回推薦改善というフィードバックループ
3. **複数ベンチマークでSOTA** - 既存LLMエージェント推薦手法を大幅に上回る性能改善

## なぜ注目？

検索・推薦エンジニアにとって重要な問題提起：**ユーザーモデルを更新するだけでなく、推論ロジック自体をユーザーごとに適応させる**という発想は、産業システムにも応用可能な新視点。LLMエージェント推薦の次の一手として注目。
