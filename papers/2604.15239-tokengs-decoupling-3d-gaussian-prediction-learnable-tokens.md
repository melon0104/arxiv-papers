# TokenGS: Decoupling 3D Gaussian Prediction from Pixels with Learnable Tokens

**Authors:** Jiawei Ren, Michal Jan Tyszkiewicz, Jiahui Huang, Zan Gojcic (NVIDIA / ETH Zürich)

**Published:** 2026-04-16

**arXiv:** https://arxiv.org/abs/2604.15239

## 概要

フィードフォワード型3DGS（3D Gaussian Splatting）予測における主要な設計選択を再考する（NVIDIA / ETH Zürich）。

従来の慣行：カメラ光線に沿った深度としてGaussianの平均を回帰 → **本論文は非最適**と主張。代わりに、自己教師ありレンダリング損失のみを使って**3D平均座標を直接回帰**する設計を提案。

これによりエンコーダ専用設計から**エンコーダ-デコーダアーキテクチャ**（学習可能なGaussianトークン）へ移行可能。

## ポイント

- **ピクセルからGaussian予測を分離**する新設計
- 学習可能なトークンによりシーン表現をより柔軟に
- NVIDIA / ETH Zürich による3DGS改善提案
