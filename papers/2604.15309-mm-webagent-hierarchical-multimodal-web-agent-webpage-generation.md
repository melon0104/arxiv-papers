# MM-WebAgent: A Hierarchical Multimodal Web Agent for Webpage Generation

**Authors:** Yan Li, Zezi Zeng, Yifan Yang, Yuqing Yang, Ning Liao

**Published:** 2026-04-16

**arXiv:** https://arxiv.org/abs/2604.15309

## 概要

AIGC（AI生成コンテンツ）ツールを活用したWebページ自動生成において、要素が孤立して生成されるため**スタイル不整合**と**グローバルコヒーレンス欠如**が問題となっていた。

本論文では **MM-WebAgent**、階層的マルチモーダルWebページ生成エージェントフレームワークを提案。ページ全体の構造を理解しながら、画像・動画・可視化を統合的に生成することでスタイル一貫性を保持する。

## ポイント

- **階層的エージェント設計**：グローバル構造把握 → ローカルコンテンツ生成の2段階
- マルチモーダル（テキスト・画像・動画・可視化）を統合したWebページ設計
- スタイル一貫性と全体コヒーレンスを自動的に維持

## 関連分野
