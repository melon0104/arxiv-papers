# TokenLight: Precise Lighting Control in Images using Attribute Tokens

**Authors:** Sumit Chaturvedi, Yannick Hold-Geoffroy, Mengwei Ren, Jingyuan Liu, He Zhang (Adobe Research)

**Published:** 2026-04-16

**arXiv:** https://arxiv.org/abs/2604.15310

## 概要

画像の再照明（relighting）において、照明の複数属性を**精密かつ連続的**に制御する手法を提案（Adobe Research）。

再照明を条件付き画像生成タスクとして定式化し、**アトリビュートトークン**（attribute tokens）で照明因子をエンコード：
- 強度（intensity）
- 色（color）
- 環境光（ambient illumination）
- 拡散レベル（diffuse level）
- 3D光源位置

大規模合成データセット（グラウンドトゥルース照明アノテーション付き）に少量の実画像を補完して訓練。

## ポイント

- **Adobe Research**による実用的なrelight技術
- 単一モデルで複数照明属性を独立・連続制御
- 属性トークンという設計で精密かつ解釈可能なコントロール
