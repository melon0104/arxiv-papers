# MLTFR: Multi-LLM Token Filtering and Routing for Sequential Recommendation

**arXiv:** https://arxiv.org/abs/2604.18200  
**カテゴリ:** cs.IR  
**投稿日:** 2026-04-20  
**コード:** https://github.com/ccwwhhh/MLTFR

## 一言サマリー

GitHub公開：テキスト入力なしでLLMのトークン埋め込みのみを活用する逐次推薦フレームワーク。複数LLMをMoE統合し、単一LLMの不安定性・限界を克服。

## 概要

LLMのトークン埋め込みを推薦に活用する研究。単一LLMのトークン埋め込みをそのまま注入すると意味的ミスアライメントや不安定な挙動を示す問題を解決。

**手法:**
- ユーザーインタラクション誘導型LLM知識統合パラダイム
- タスク関連トークン埋め込みをユーザーガイドフィルタリングで選択（ノイズ除去）
- Mixture-of-Experts (MoE)アーキテクチャで複数LLMトークン空間を統合
- Fisher加重セマンティック合意エキスパートで異種エキスパート間の支配を防止

**結果:** 最先端の逐次推薦ベースラインと既存アライメント手法を一貫して上回る。

## キーポイント

- テキストコーパスやバックボーン改変不要
- 複数LLMの相補的知識を効率的に統合
- コーパスフリーかつ安定した推薦改善を実証
