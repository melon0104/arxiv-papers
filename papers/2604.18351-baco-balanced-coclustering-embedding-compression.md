# BACO: Balanced Co-Clustering of Users and Items for Embedding Table Compression in Recommender Systems

**arXiv:** https://arxiv.org/abs/2604.18351  
**カテゴリ:** cs.IR  
**投稿日:** 2026-04-20  
**採択:** SIGIR 2026

## 一言サマリー

産業規模の推薦システムで埋め込みテーブルを75%以上圧縮しながら精度低下わずか1.85%を実現、最強ベースライン比346倍高速な協調フィルタリングベースのco-clustering手法。

## 概要

推薦システムの深層学習モデルは、ユーザー・アイテムごとの高密度埋め込みベクトルを持つが、産業スケールではメモリ・計算コストが膨大になる。BACOはユーザー・アイテム間の協調信号（ユーザーインタラクション）を利用してグルーピングし、同一グループのユーザー/アイテムは埋め込みを共有する（コードブック方式）。

**主な貢献:**
- イントラクラスター接続性最大化とクラスターボリューム均衡を同時に満たす均衡co-clusteringの定式化
- 正規グラフクラスタリング技術を理論的に統合
- 重み付けスキーム + ラベル伝播ソルバー + 二次ユーザークラスター

**結果:** ベンチマーク全体で埋め込みパラメータを75%以上削減、recall低下は最大1.85%、最強ベースライン比で最大346倍高速。

## キーポイント

- ID hashingに代わる協調信号ベースのアプローチ
- コードブックcollapse防止のための均衡制約
- SIGIR 2026採択で産業応用性が高い
