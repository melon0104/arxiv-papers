# T-REN: Learning Text-Aligned Region Tokens for Dense Vision-Language Alignment

**arXiv:** https://arxiv.org/abs/2604.18573  
**カテゴリ:** cs.CV  
**投稿日:** 2026-04-20  
**コード:** https://github.com/savya08/T-REN

## 一言サマリー

GitHub公開：ビジョンバックボーンに3.7%パラメータ追加のみで、テキスト整合リージョントークンを学習—セマンティックセグメンテーション+5.9mIoU、動画トークン数を187倍削減。

## 概要

Vision-Language Encoderの2つの課題：(1) 言語とdenseビジョン特徴間の弱いアライメント（オープン語彙セグメンテーション精度低下）、(2) 高トークン数（長動画スケーラビリティ制限）を同時解決。

**手法（T-REN: Text-aligned Region Encoder Network）:**
- 凍結ビジョンバックボーン上の軽量ネットワーク
- 各セマンティック領域内のパッチ表現をリージョントークンに集約
- リージョンレベルテキストアノテーションとアライメント

**結果:**
- ADE20K open-vocab segmentation: +5.9 mIoU
- COCO object-level retrieval: +18.4% recall
- Ego4D video object localization: +15.6% recall
- VSPW video scene parsing: +17.6% mIoU
- 画像トークン数24倍削減、動画187倍削減

## キーポイント

- 極めて軽量な追加コストで大幅な改善
- 動画への大規模スケーリングを可能に
- dense理解と効率性の両立
