# Sessa: Selective State Space Attention

**arXiv:** https://arxiv.org/abs/2604.18580  
**カテゴリ:** cs.LG, cs.AI, cs.CL  
**投稿日:** 2026-04-20  
**コード:** https://github.com/LibratioAI/sessa

## 一言サマリー

GitHub公開：Attention をフィードバックパス内に配置し「多経路再帰集約」を実現。長距離依存性がべき乗則的に減衰するSessaはTransformer・Mambaを長文脈ベンチマークで凌駕。

## 概要

Transformerは拡散アテンションで古いトークンの影響が1/Seff(t)に希釈。Mambaなど選択的SSMは入力依存フィードバックだが長区間では感度が指数減衰。Sessaはアテンションをフィードバックパス内に配置し両者を統合。

**理論的特性:**
- lag lに対して O(l^{-β}) (0<β<1) べき乗則メモリテールを実現（1/lより遅く減衰）
- 非減衰プロファイルを含む柔軟な選択的検索を実現するのはSessaのみ

**結果:** 長文脈ベンチマークで最強の性能、短文脈言語モデリングでもTransformer・Mamba互角。

## キーポイント

- 長距離依存性の理論的保証を持つ新アーキテクチャ
- Transformer・Mambaの両方の弱点を統合的に克服
- コード公開で再現性高
