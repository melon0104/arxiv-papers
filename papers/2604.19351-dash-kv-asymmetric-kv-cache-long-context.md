# DASH-KV: Accelerating Long-Context LLM Inference via Asymmetric KV Cache Hashing

**arXiv:** https://arxiv.org/abs/2604.19351  
**Authors:** Jinyu Guo, Zhihan Zhang, Yutong Li, Jiehui Xie, Md. Tamim Iqbal  
**Submitted:** 2026-04-21

## 概要

長文脈LLM推論のKVキャッシュ圧縮手法 **DASH-KV** を提案。既存のKVキャッシュ圧縮は品質低下や浮動小数点演算オーバーヘッドの課題があったが、非対称ハッシングを用いることで重要なKVエントリを高速に識別・保持し、推論を大幅に高速化する。

## 選定理由

- LLM長文脈推論の効率化という重要テーマ
- 実装可能なKVキャッシュ最適化

## キーポイント

- **問題**: 標準アテンションは文脈長に対して2乗コスト、KVキャッシュが巨大化
- **解決策**: 非対称ハッシング（DASH）でKVエントリの重要度を効率的に推定
- **効果**: 生成品質を保ちながら推論を高速化、メモリ使用量を削減
- 既存のKVキャッシュ圧縮手法より精度・速度のバランスが優秀
