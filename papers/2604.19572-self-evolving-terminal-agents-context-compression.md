# A Self-Evolving Framework for Efficient Terminal Agents via Observational Context Compression

**arXiv:** https://arxiv.org/abs/2604.19572  
**Authors:** Jincheng Ren, Siwei Wu, Yizhi Li, Kang Zhu, Shu Xu  
**Submitted:** 2026-04-21

## 概要

長期・多ターンのターミナルエージェントタスクにおける**トークンコスト爆発問題**を解決する自己進化フレームワーク。環境フィードバックを反復的に保持すると履歴がステップ数の2乗で成長するが、観測コンテキストの圧縮で重要情報を保持しつつコストを線形以下に抑える。

## 選定理由

- LLMエージェントの実用化における根本的課題（コンテキスト長爆発）への解答
- 自己進化（継続学習）の要素を持つ

## キーポイント

- **問題**: ターミナルエージェントは多ターンで環境フィードバックが蓄積 → トークンコスト2乗増加
- **解決策**: 観測コンテキスト圧縮 + 自己進化（タスク完了経験から効率的圧縮戦略を学習）
- **効果**: トークンコストを大幅削減しながら性能維持、長期エージェントタスクで有効
- SWEBench等のエージェントベンチマークで検証
