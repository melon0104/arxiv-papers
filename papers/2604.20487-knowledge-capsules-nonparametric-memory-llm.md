# Knowledge Capsules: Structured Nonparametric Memory Units for LLMs

**arXiv:** https://arxiv.org/abs/2604.20487  
**Submitted:** 2026-04-22  
**Authors:** Bin Ju, Shenfeng Weng, Danying Zhou, Kunkai Su

## 概要

LLMのパラメトリック重みに知識を埋め込む代わりに、**Knowledge Capsules**という構造化された非パラメトリックメモリ単位を提案。RAGの単純なチャンク検索を超え、カプセル単位で知識を管理・更新・検索できるアーキテクチャ。

## 手法のポイント

- 知識を「カプセル」単位（エンティティ+関係+文脈）で構造化保存
- カプセルは更新・削除が容易→知識の鮮度を維持
- 検索時はカプセル単位で最適な粒度の知識を取得

## なぜ重要か

RAGの課題である「チャンク境界問題」「知識更新コスト」を同時に解決しようとするアプローチ。LLMのメモリ外部化の新パラダイムとして注目。

## 選定理由

RAG/LLMメモリの革新的アーキテクチャ提案。知識管理の効率化に直結するcs.CL論文。
