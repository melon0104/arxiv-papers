# Learning to Evolve: A Self-Improving Framework for Multi-Agent Systems via Textual Parameter Graph Optimization

**arXiv:** https://arxiv.org/abs/2604.20714  
**Submitted:** 2026-04-22  
**Authors:** Shan He, Runze Wang, Zhuoyun Du, Huiyu Bai

## 概要

マルチエージェントシステム（MAS）の設計・最適化を「Agent Engineering」と定義し、その自動化フレームワークを提案。テキストパラメータグラフ（エージェントの役割・プロンプト・連携構造をグラフとして表現）を最適化することで、MASが自己改善できる仕組みを構築。

## 技術的貢献

- エージェント間の連携構造をグラフとして形式化
- テキストレベルの最適化（プロンプト、役割割り当て）を勾配フリー手法で実施
- ベンチマーク上でのMAS自動設計で既存手法を上回る

## なぜ重要か

MASの手動設計コストは産業展開の大きな障壁。自己改善フレームワークはエージェントオーケストレーションの未来を示す。

## 選定理由

マルチエージェント×自己改善という2026年最重要テーマを扱う。実用的かつ研究的貢献の両立。
