# SWE-chat: Coding Agent Interactions From Real Users in the Wild

**arXiv:** https://arxiv.org/abs/2604.20779  
**Submitted:** 2026-04-22  
**Authors:** Joachim Baumann, Vishakh Padmakumar, Xiang Li, John Yang et al.

## 概要

AIコーディングエージェントが実際のユーザーにどう使われているかを大規模に研究した初めての論文。**SWE-chat**：実ユーザーとコーディングエージェントの会話ログを収集・分析したデータセットとその分析。エージェントの出力がどれだけ実際に役立つかを定量評価。

## 主な発見

- ユーザーはエージェントの提案の約X%を採用（詳細はarXiv参照）
- タスク分類：バグ修正・機能追加・説明・リファクタリングの分布
- エージェントが失敗するパターンの体系的分析
- 人間とエージェントのコラボレーションダイナミクスの可視化

## なぜ重要か

エージェント研究はベンチマーク（SWE-bench等）中心だったが、**実ユーザー使用データ**は希少。産業レベルのコーディングエージェント開発の必須参照資料となる。

## 選定理由

「Real-world」のAIエージェント評価という貴重なデータポイント。産業的関連性が極めて高い。
