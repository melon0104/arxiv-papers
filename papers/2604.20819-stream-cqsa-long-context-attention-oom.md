# Stream-CQSA: Avoiding Out-of-Memory in Attention Computation via Flexible Workload Scheduling

**arXiv:** https://arxiv.org/abs/2604.20819  
**Submitted:** 2026-04-22  
**Authors:** Yiming Bian, Joshua M. Akey

## 概要

長文コンテキストLLMの根本的制約である**Self-Attentionの二乗メモリコスト**によるOOM（Out-of-Memory）問題を解決。Stream-CQSA（Chunked Query Streaming Attention）はワークロードを柔軟にスケジューリングすることで、メモリ増大を抑えながら正確なアテンション計算を維持。

## 手法のポイント

- クエリを小チャンクに分割してストリーミング処理
- Key/Valueは全体を保持しつつ、Queryのメモリフットプリントを削減
- 既存FlashAttentionと異なりより柔軟なワークロード分割を実現

## なぜ重要か

100K〜1M tokenの長文脈LLM推論における実用上の最大課題の一つ。産業デプロイの実現可能性を向上させる純粋な工学的貢献。

## 選定理由

長文脈LLMのインフラ最適化という産業直結テーマ。実装可能性が高い。
