# Parallel-SFT: Improving Zero-Shot Cross-Programming-Language Transfer for Code RL

**arXiv:** https://arxiv.org/abs/2604.20835  
**Submitted:** 2026-04-22  
**Authors:** Zhaofeng Wu, Shiqi Wang, Boya Peng, Anuj Goyal (Amazon)

## 概要

コードLLMのRLトレーニング（RLVR）において、Python等の高リソース言語で学習したモデルがニッチな言語（低リソースPL）に転移しにくい問題を解決。Parallel-SFTは、RL最適化と並行してSFTを実施することで、言語間の知識を安定的に保持・転移させる手法。

## 技術的貢献

- RL学習中の「言語忘却」現象を実証的に分析
- Parallel-SFT: RL更新と並行したSFTで安定した多言語コード能力を維持
- AmazonのCode評価ベンチマークで検証

## なぜ重要か

GitHub Copilot等のコード補完システムは多言語対応が必須。RLによるコード品質向上と多言語カバレッジの両立は産業上の重要課題。Amazon研究者による成果。

## 選定理由

Amazonによるコード生成LLMの産業的研究。Code RL×多言語転移の実用的貢献。
