# Pre-trained LLMs Meet Sequential Recommenders: Efficient User-Centric Knowledge Distillation

**arXiv:** https://arxiv.org/abs/2604.21536
**公開日:** 2026-04-23
**カテゴリ:** cs.IR, cs.CL

## 著者
Nikita Severin, Danil Kartushov, Vladislav Urzhumov

## 概要（日本語）
逐次推薦システム（Sequential Recommender）は時系列ユーザー行動の学習に優れるが、インタラクションパターン以上のリッチなユーザー意味論を捉えることが難しい。本研究ではLLMのリッチな表現を逐次推薦モデルへ効率的に蒸留するユーザー中心の知識蒸留フレームワークを提案。推論時のLLMコスト不要で、LLMの豊かな意味理解を推薦品質向上に活用できる。

## ポイント
- 推論時LLM不要の効率的な知識蒸留
- ユーザー意味論をLLMから逐次推薦モデルへ転移
- Eコマース・メディア推薦に実用的なアプローチ

## Abstract
Sequential recommender systems have achieved significant success in modeling temporal user behavior but remain limited in capturing rich user semantics beyond interaction patterns. Large Language Models offer rich semantic representations but are computationally expensive at inference time. We propose an efficient user-centric knowledge distillation framework that transfers LLM knowledge into sequential recommenders during training, eliminating inference-time LLM costs while improving recommendation quality through richer semantic understanding.
