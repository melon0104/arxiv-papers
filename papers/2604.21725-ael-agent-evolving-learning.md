# AEL: Agent Evolving Learning for Open-Ended Environments

**arXiv:** https://arxiv.org/abs/2604.21725  
**Published:** 2026-04-23  
**Authors:** Wujiang Xu, Jiaojiao Han, Minghao Guo, Kai Mei  
**Venue:** ACL関連  
**Code:** GitHub公開済み

## 概要

LLM エージェントはオープンエンドな環境（数百エピソードに渡るシーケンシャルタスク）でほぼステートレスに動作しており、過去経験を将来の行動改善に活かせていない。AEL は「何を覚えるか」でなく「記憶をどう使うか」に焦点を当て、適切な検索ポリシー・過去経験の解釈・改善への転換を実現する学習フレームワーク。

## 主要貢献

- エージェントが長期エピソードから継続的に学習する仕組み
- 過去経験の検索・解釈・活用の統合設計
- 複数のオープンエンドベンチマークで SOTA
- コード公開

## 関連分野

LLMエージェント・継続学習・強化学習

## ポイント

> エージェントの「記憶活用」問題に正面から取り組む研究。RAG的な記憶取得だけでなく、学習ループに組み込む点が新しい。
