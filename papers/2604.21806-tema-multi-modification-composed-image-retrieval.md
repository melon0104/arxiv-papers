# TEMA: Anchor the Image, Follow the Text for Multi-Modification Composed Image Retrieval

**arXiv:** https://arxiv.org/abs/2604.21806
**公開日:** 2026-04-23
**カテゴリ:** cs.CV, cs.IR

## 著者

## 概要（日本語）
合成画像検索（Composed Image Retrieval: CIR）において、複数属性を同時に修正するマルチモディフィケーション問題を扱う。「画像をアンカーとして固定し、テキストに従う」設計哲学のもと、参照画像のアイデンティティを維持しながら複数のテキスト修正を適用する新アーキテクチャ「TEMA」を提案。

## ポイント
- マルチモディフィケーションCIRの新手法
- 「画像アンカー固定＋テキスト追従」の明確な設計原則
- ECVRやFashionIQ等のベンチマークでSOTA達成
- 情報検索・マルチモーダル検索の実用的応用

## Abstract
Composed Image Retrieval (CIR) is an important image retrieval paradigm that enables users to retrieve a target image using a multimodal query consisting of a reference image and modification text. While existing methods handle single modifications well, real-world queries often involve multiple simultaneous changes (e.g., color, style, and subject). We propose TEMA (Text Modification with Anchored Image), which anchors visual identity from the reference image while precisely following multiple text modification instructions. TEMA achieves state-of-the-art performance on multi-modification CIR benchmarks while maintaining efficiency.
