# UniGenDet: A Unified Generative-Discriminative Framework for Co-Evolutionary Image Generation and Generated Image Detection

**arXiv:** https://arxiv.org/abs/2604.21904  
**Published:** 2026-04-23  
**Authors:** Yanran Zhang, Wenzhao Zheng, Yifei Li, Bingyao Yu  
**Code:** GitHub公開済み

## 概要

画像生成と生成画像検出は急速に発展しているが、これまで独立したアーキテクチャで研究されてきた（生成：生成ネットワーク、検出：識別ネットワーク）。**UniGenDet** は生成と検出を単一フレームワークに統合し、対抗的情報を共有することで双方が共進化する統一モデルを提案。

## 主要貢献

- 生成・検出タスクの共進化フレームワーク
- 対抗的学習による双方向の性能向上
- フェイク画像検出の汎化性能改善
- コード公開

## 関連分野

コンピュータビジョン・画像生成・フェイク検出
