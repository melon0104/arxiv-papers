# Long-Horizon Manipulation via Trace-Conditioned VLA Planning (LoHo-Manip)

**arXiv:** https://arxiv.org/abs/2604.21924
**公開日:** 2026-04-23
**カテゴリ:** cs.RO, cs.AI

## 著者
Isabella Liu, An-Chieh Cheng, Rui Yan

## 概要（日本語）
VLA（Vision-Language-Action）ポリシーによる長期ロボット操作の課題に取り組む研究。実タスクはマルチステップで進捗依存、かつ誤差が累積するため単一VLAでは限界がある。本研究「LoHo-Manip」は**タスク管理VLM**（高レベル）と**実行VLA**（低レベル）を分離し、Receding-Horizon方式で長期操作を実現する。

## ポイント
- VLMとVLAの階層分離でロボット長期操作を解決
- Receding-Horizonプランニングで進捗依存タスクに対応
- 誤差累積問題を階層設計で軽減
- ロボット操作の実用性向上に直結

## Abstract
Long-horizon manipulation remains challenging for vision-language-action (VLA) policies: real tasks are multi-step, progress-dependent, and brittle to compounding execution errors. We present LoHo-Manip, a hierarchical framework that decomposes long-horizon manipulation into task management and execution. A high-level VLM generates trace-conditioned plans that guide a low-level VLA policy executing individual primitives. Using a Receding-Horizon approach, the system continuously re-plans based on execution progress, enabling recovery from failures and adaptation to environmental changes across extended manipulation sequences.
