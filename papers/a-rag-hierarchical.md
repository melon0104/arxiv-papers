# A-RAG: Scaling Agentic Retrieval-Augmented Generation via Hierarchical Retrieval Interfaces

## 概要
フロンティア言語モデルの推論・長期ツール使用能力を活用したエージェント型RAGシステム。階層的検索インターフェースでスケーラビリティを実現。

## 主要貢献
- **既存RAGの限界指摘**: モデルが検索決定に参加できない設計の問題
- 階層的検索インターフェースの導入
- テスト時計算量に応じた効率的スケーリング

## 技術詳細
- 単発検索やワークフロー固定からの脱却
- エージェントが検索戦略を動的に決定
- 複数抽象度レベルでの検索操作
