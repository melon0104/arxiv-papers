# Understanding Agent Scaling in LLM-Based Multi-Agent Systems via Diversity

## 概要
LLMベースのマルチエージェントシステム（MAS）におけるスケーリングの限界と、異種性（多様性）がもたらすゲインを分析。

## 主要貢献
- **スケーリングの収穫逓減発見**: 同種エージェント増加の限界を定量化
- 異種性（異なるモデル、プロンプト、ツール）によるゲイン継続
- 多様性がスケーリングを可能にするメカニズム解明

## 技術詳細
- 同種設定でのスケーリング収穫逓減の実証
- 異種性導入による継続的なゲインの検証
- MASスケーリングの理論的基盤
