# CM2: Reinforcement Learning with Checklist Rewards for Multi-Turn Tool Use

[📄 arXiv:2602.12268](https://arxiv.org/abs/2602.12268) | [GitHub](https://github.com/namezhenzhang/CM2-RLCR-Tool-Agent)

## 概要
マルチターン・マルチステップのツール使用エージェント向けRLフレームワーク。検証可能な報酬を持たないオープンエンドな振る舞いに対応。

## 課題認識
- 現実的な目的は検証可能な報酬を欠く
- マルチターン・マルチステップのエージェントRLは未開拓
- 実行可能ツール環境の構築・維持はコスト高

## 提案手法
**CM2**：
- **チェックリスト報酬**で検証可能な結果報酬を代替
- 各ターンの意図された振る舞いを細粒度バイナリ基準に分解
- 明示的なエビデンスグラウンディング＋構造化メタデータ
- オープンエンド判定を安定した分類スタイル判定に変換
- スパース報酬割り当て＋密な評価基準でバランス
- LLMシミュレートツール環境でスケーラブルな訓練

## 実験結果（8B Baseモデル、8k例RL）
- τ-Bench: SFT比**+8pt**
- BFCL-V4: **+10pt**
- ToolSandbox: **+12pt**
- 同規模オープンソースベースライン（判定モデル含む）と同等以上
