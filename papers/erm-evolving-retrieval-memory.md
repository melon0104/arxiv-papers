# ERM: Evolving Retrieval Memory - RAG without Forgetting

- **arXiv**: [2602.05152](https://arxiv.org/abs/2602.05152)
- **カテゴリ**: cs.IR
- **投稿日**: 2026-02-05

## 概要

RAGシステムにおいて、クエリ時の適応を一時的ではなく持続的な検索改善に変換するトレーニング不要のフレームワーク「Evolving Retrieval Memory (ERM)」を提案。

## 主要な貢献

- **正解ゲート付きフィードバック**: 正解した拡張シグナルのみを選択的にインデックスに反映
- **安定した更新**: ノルム制限付きで段階的にキーを進化
- **理論的保証**: クエリ拡張とキー拡張の等価性を証明、選択的更新の収束を保証

## 実験結果

- BEIRとBRIGHTの13ドメインで一貫した改善
- 推論時オーバーヘッドなしでネイティブ検索速度を維持
- 特に推論集約型タスクで顕著な効果
