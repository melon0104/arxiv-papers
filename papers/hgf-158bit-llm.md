# Hybrid Gated Flow (HGF): Stabilizing 1.58-bit LLMs via Selective Low-Rank Correction

- **arXiv**: [2602.05269](https://arxiv.org/abs/2602.05269)
- **カテゴリ**: cs.LG, cs.AI, cs.CL
- **投稿日**: 2026-02-05

## 概要

LLMのエッジデバイス展開において「メモリウォール」問題に対処。1.58ビット量子化の品質劣化を回復するデュアルストリームアーキテクチャ「Hybrid Gated Flow (HGF)」を提案。

## 問題背景

- 1.58ビット量子化（BitNet b1.58）: メモリ削減は劇的だが、FP16比でパープレキシティ20-25%劣化
- メモリ帯域幅がボトルネック（計算ではなく）

## 主要な貢献

- **デュアルストリームアーキテクチャ**: 1.58ビット三値バックボーン + 学習可能な低ランクFP16補正パス
- **適応ゲート制御**: 補正パスを動的に制御

## 実験結果

- TinyStoriesデータセットでの実験
- HGF 5.4: 検証損失0.9306 (BitNet: 1.0294)
- FP16ベースライン(0.8490)との品質ギャップの**約55%を回復**
- 追加メモリオーバーヘッドは**わずか12-15%**

## 発見

**量子化は構造的正則化として機能**: フルプレシジョンのdifferential attentionベースラインは訓練不安定（損失1.68超）だが、三値アンカーHGFは安定収束

## スケール検証

1.2B、3BパラメータモデルでSlimPajama、FineWeb-Eduを使った実験でも線形スケールを確認
