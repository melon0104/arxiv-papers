# HUSKY: Humanoid Skateboarding System via Physics-Aware Whole-Body Control

## 概要
ヒューマノイドロボットによるスケートボード操作。非ホロノミック制約下での動的全身制御を実現。

## 主要貢献
- **高難易度タスク達成**: 劣駆動の車輪付きプラットフォーム上での安定制御
- 物理認識型全身制御フレームワーク
- 緊密なヒューマン・オブジェクトインタラクション

## 技術詳細
- 静的環境仮定を超えた動的タスク
- 非ホロノミック制約下での操縦戦略
- ヒューマノイドの動的バランス制御

🛹 ヒューマノイド制御の新境地
